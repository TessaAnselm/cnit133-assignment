Part I:

Create a web page for part I with:

  • A heading.
  • Hyperlink to go back to the main Homework 5 page.
  • A form that will contain:
        An input text box so the user will enter his/her full name.
        A group of radio buttons so the user will select his age group (less than 21; between 21 and 50; older than 50.).
        A group of checkboxes so the user will select the browsers he/she has used (Firefox, Edge, Chrome, Safari).
        A select element or datalist (to produce a drop-down list of choices) so the user will select which type of movie he/she prefers (Action, Comedy, Drama, Documentary, Science Fiction).
  • A button to submit the form.
  • A button to clear the form.
  • A div or paragraph element that will serve as the output/result area.
  • Code a script that will validate whether the user has done the following:
        Entered data in the input text box.
        Checked one of the radio buttons.
        Checked at least one checkbox
        Selected an option from the list of options in the select (or datalist) element.
        If one of the data, from the user, is missing, you will show an error message in the output area.
        If ALL the requested data was input by the user, you will write a message in the output area such as "Thanks, your data was submitted!"

Part II:

Create a web page for part II with:

  • A heading.
  • Hyperlink to go back to the main Homework 5 page.
  • Two pulldown menus:

Part III:

The web page for part 3 and the script will be based on the table you see below that shows data about several of our states from the July/2019 census!

Create a web page for part II with:

  • A heading.
  • Hyperlink to go back to the main Homework 5 page.
  • A button to submit the data.
  • A button to clear the form.
  • A form with an input text box - the user will use that input box to type the full name of the state or the two-letter abbreviation of the state.
  • A textarea inside the form or a different element (div or paragraph) where you will write the output/result.

You will write a script that:

  • The user will type the FULL NAME or ABBREVIATION of the state he/she desires to get information.
    After the user click the button to get "State Info", you will verify if the state typed by the user exists in your database.
  • If the state does not exist, write a message in the output area saying something as "Sorry, we do not have information about this state! We only have information about....."
  • If the state exists, you should present in the output/result area the information from the state required.
