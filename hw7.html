Homework 7 Instructions on Topic "Objects"

What you will do:

Create a main web page for Homework 7 with the following:
  • Heading "CNIT 133 Homework 7 - Object Model and Event Model".
  • The jQuery script library in the head part of your pages.
  • Hyperlink and a short summary of that part.
  • Hyperlink to go back to the main page of the course (Homework 1 web page).

Part I:

Create a web page with:

  • A heading of your choice.
  • A div, or a main tag with at least 3 paragraphs and that will be the area to be used to apply the styles chosen by the user.
  • A form that will have:
        A radio buttons for the user to select the color of the background to be applied.
        A checkboxes that will allow the user to choose "underline", and/or "bold", and/or "italic".
        A select element to create a drop-down list where you will allow the user to select the font-size to be applied.
        No button to clear or submit the choices.
        When the user selects a style, that style will be applied right away.
  • A hyperlink so the user can go back to the page of Homework 7.
  • A script (it can be in JavaScript or jQuery) that will do the following:
        As the user starts selecting the styles available, apply selected style to the area mentioned above.
        When user unchecks from checked, you will need to remove the style that was unchecked from area.
        The styles should be added as the user keeps selecting/checking the styles from the form you provided.


Part II:

Create a web page for part II with:

  • A heading of your choice.
  • A hyperlink so the user can go back to the page of Homework 7.
  • A navigation with a link to Homework 2, Homework 3, and Homework 4.
  • You will code a script that when the user mouseover on any of those 3 items of your navigation, you will display a drop-down menu that contains the links of the parts of each Homework (part 1, part 2, etc.).
