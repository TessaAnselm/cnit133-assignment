Homework 6 Instructions on Topic "Objects"

What you will do:

Create a main web page for Homework 6 with the following:
  • Heading CNIT 133 Homework 6 - Objects.
  • Hyperlink to Part 1 and a short summary of that part.
  • Hyperlink to Part 2 and a short summary of that part.
  • Hyperlink to Part 3 and a short summary of that part.
  • Hyperlink to go back to the main page of the course (Homework 1 web page).

Part I:

Create a web page with:

  • A heading that you can choose.
  • A form that will have:
        An input box for the user to enter a number with 4 decimals.
        A button to submit the data.
        A button to clear the form.
        A textarea (or another element outside the form - div or paragraph) where you will write the output/result.
        A hyperlink to go back to the main page of Homework 6.
  • If the user types a valid number (with 4 decimals and it is a number), you will use Math.round, Math.sqrt, Math.floor, toFixed() to obtain:
        A. Round the floating-point number to the nearest integer.
        B. Calculate the square root of the floating-point number and round it to an integer.
        C. Round the floating-point number to the nearest tenths position.
        D. Round the floating-point number to the nearest hundredths position.
        E. Round the floating-point number to the nearest thousandths position.
  • Present in the output/result area the original number typed by the user and the results you found for each calculation requested above.

Part II:

Create a web page for part II with:

  • A heading that you can choose.
  • A form with:
        A textarea to allow the user to type a long content.
        An input box to allow the user to type only one letter.
        An input box or textarea, or an element outside the form (a div or paragraph) to present the output/result.
        A hyperlink to go back to the main page of Homework 6.

Write a script using some Script methods you learned, to solve the following:

  • The user will type whatever content he/she wants in the textarea for the long content, and the user will also type a single character in the input box you provided.
    When the user clicks the submit button, you will:
        Search for the character the user typed inside the content in the textarea and you will count how many times this character is shown in the big content.
        If the character is found, you will present a message in the output/result area that says how many times the character X
        shows up in the content where X is the single character typed by the user.
        If the character is not found, you will present a new window (using the window.open() method you learned), the message: "Search character X not found
        in the content you typed", where X is the single character typed by the user.

Part III:

Create a web page for part II with:

  • A heading that you can choose.
  • An input box for the user to type the phone number.
  • 3 input boxes where you will present the result/output (area code, the first 3 digits of the phone number, the last 4 digits of the phone number).
  • A button to start the process.
  • A a button to start the process.
  • An element such as div or paragraph to present any error message you need.
  • A hyper link to go back to the main page of Homework 6.

Create a script to resolve the following:

  • The user needs to type the telephone number, as a string, in this format (999) 999-9999.
  • Use the String method split or the match group related to regex to extract the area code; the first 3 digits of the phone number; and the last 4 digits of the
    phone number.
  • Display in the 3 input boxes (that are read-only) you provided in the form, those separated numbers - ONLY THE NUMBERS, there should be no parentheses, no hyphen!
  • If the user does not type the phone number in the exact format requested, you should display in the other element (div or paragraph) an error message that clarifies
    to the user how he should type the phone number and then, NONE OF THE 3 READ-ONLY INPUT BOXES SHOULD PRESENT A RESULT.
  • Use jQuery for extra credit.
